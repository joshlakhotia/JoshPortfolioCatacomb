PROJECTS


☪ Does It Go

My main project. A full stack PERN database app that displays weather conditions for multiple paragliding launch points on one page. This eliminates the need to hop to several different forecasts to find a flyable launch.

☪ The Catacombs

The very site you are on right now! Made as a little test of basic HTML and CSS skills. I went with a retro video game vibe bacause thats the look I enjoy most and I think is super unique, especially in today's world of modern websites.

☪ Weather Flow Field

A p5.js project that takes the current conditions for whatever location you input into the api and returns a flow field that uses those conditions to adjust the flow filed's parameters. Temperature controls color, wind controls flow angle intensity, etc.
